# SamsungPixelOS v1.0-beta (SM-X110 / gta9wifi)

Pixel-style GSI package layout for Samsung Galaxy Tab A9 WiFi.

## Device target

- **Model:** `SM-X110`
- **Codename:** `gta9wifi`
- **Platform:** MediaTek `mt8781`
- **Dynamic partitions:** enabled

## Included asset groups

- Recovery package(s)
- System image/package
- VBMeta package(s)
- Optional source patch bundle reference

## Integrity verification

All SHA-256 checksums are documented in:

- `flash-assets/release-manifest.md`

Please verify checksums before flashing.

## Flashing notes

- This project is organized for GSI-style workflows.
- Use image combinations that match your method (Odin tar workflow vs image workflow).
- Incorrect combinations can cause bootloops or non-boot.

See:

- `FLASHING.md`
- `releases/minimal-upload-set.md`

## Known caveats

- This release layout does not guarantee every image combination boots on every setup.
- Device-specific behavior can vary by firmware revision.
- Always keep rollback/stock firmware options available.
